# EU trade-dependency pipeline (Comext)

Public-data demo: extra-EU import dependency for a chosen product, comparing a
naive supplier ranking (by trading partner) against an origin-corrected ranking
to remove the Rotterdam/Antwerp quasi-transit distortion. First instance:
rare-earth permanent magnets (CN 8505 11 10).

## Layout
- comext-magnet-dependency-demo.R   pipeline scaffold
- raw/   downloaded Comext data (gitignored)
- out/   generated datasets and charts (gitignored)

Public Eurostat data only.
